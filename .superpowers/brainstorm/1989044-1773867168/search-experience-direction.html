Search Experience — Design Direction

What vibe should the search feel like for your buyers?

A

Command Center

Dense, data-rich dashboard feel. Source status indicators, live connector feeds, split-panel results. Think Bloomberg terminal meets sourcing — maximum information density for power users who live in this tool all day.

B

Clean & Focused

Google-style simplicity — big search bar, clean result cards, progressive disclosure. Results stream in as connectors respond. Filters are tucked away until needed. Prioritizes scannability and speed over density.

C

Hybrid — Clean Entry, Rich Results

Starts minimal (big search bar, recent searches). Once results arrive, transitions to a richer layout — source progress bar at top, sortable table with inline expansion, connector health sidebar. Progressive complexity that matches the user's journey.

Context

Your current search: single input → flat list of lead cards with confidence badges, source filters, and a detail drawer. 8 connectors return results in parallel but the UI waits for all of them before showing anything.
