# Acceptance Checklist

The recovery is acceptable when:

## Layout
- Sidebar no longer crowds or overlaps the page content.
- Main working area is visually dominant.
- Filters and controls have enough space to breathe.
- Results/list pane is comfortably readable.
- Detail area is used properly when an item is selected.

## Interaction
- Empty state appears only when nothing is selected.
- Selecting an item loads meaningful detail content.
- No resizable split-pane behavior is introduced.

## Product constraints
- No Buyer Follow-Up Queue.
- Confidence and Safety remain separate concepts where sourcing leads are shown.
- Source attribution remains visible.
- Suggested next action remains visible.
- Vendor safety language remains caution-oriented.

## Visual quality
- No obvious overflow glitches.
- No awkward nested scroll behavior.
- Page feels intentional, not crammed.
