# Key Decisions

## Confirmed decisions
1. No resizable split-pane sourcing layout.
   - The wireframes and design docs support list → detail interaction, not resizable panes.

2. No Buyer Follow-Up Queue.
   - This feature was intentionally dropped due to expected noise and low user value.

3. Confidence and Safety remain separate.
   - Lead Confidence = stock likelihood
   - Vendor Safety = trust/risk for outreach

4. One lead per vendor per part.
   - Multiple evidence items can support that lead.

5. AI/web search is augmentation only.
   - It can discover or enrich leads, but should not outrank stronger direct evidence by itself.

6. Vendor safety language must be caution-oriented.
   - Good: "Verify identity before engaging"
   - Bad: "Scammer", "Fraudster"
