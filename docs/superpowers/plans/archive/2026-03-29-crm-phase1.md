# CRM Phase 1: Shell + Customer Sales Workspace — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Vendors, Customers, and My Vendors bottom nav tabs with a single CRM tab containing a Customers | Vendors tab bar, and add staleness indicators to the customer list.

**Architecture:** New `app/routers/crm/views.py` sub-router provides the CRM shell partial. The shell template uses Alpine.js tabs with HTMX lazy-loading to embed existing customer and vendor list partials inside `#crm-tab-content`. Staleness is computed from `Company.last_activity_at` in Python — no query changes needed. Vendor list template is parameterized to accept `hx-target` overrides so it stays inside the CRM shell.

**Tech Stack:** FastAPI, Jinja2, HTMX 2.x, Alpine.js 3.x, Tailwind CSS

**Spec:** `docs/superpowers/specs/2026-03-29-crm-phase1-design.md`

"""CRM shell views — unified CRM tab with Customers/Vendors sub-tabs.

Called by: app/routers/crm/__init__.py (included via router)
Depends on: app/dependencies (require_user), app/templates
"""

from fastapi import APIRouter, Depends, Request
from fastapi.responses import HTMLResponse

from ...dependencies import require_user
from ...models.auth import User

router = APIRouter(tags=["crm"])


@router.get("/v2/partials/crm/shell", response_class=HTMLResponse)
async def crm_shell(
    request: Request,
    user: User = Depends(require_user),
):
    """Render the CRM shell with Customers/Vendors tab bar."""
    from ...main import templates

    ctx = {
        "request": request,
        "user": user,
        "default_tab": "customers",
    }
    return templates.TemplateResponse("htmx/partials/crm/shell.html", ctx)
```

STALENESS_OVERDUE_DAYS = 30
STALENESS_DUE_SOON_DAYS = 14


def _staleness_tier(last_activity_at):
    """Compute staleness tier from last_activity_at timestamp."""
    if last_activity_at is None:
        return "new"
    from datetime import datetime, timezone
    days = (datetime.now(timezone.utc) - last_activity_at).days
    if days >= STALENESS_OVERDUE_DAYS:
        return "overdue"
    if days >= STALENESS_DUE_SOON_DAYS:
        return "due_soon"
    return "recent"
```
