# Scope and Non-Goals

## In Scope
- sourcing results list
- lead detail experience
- source attribution
- confidence scoring presentation
- vendor safety review presentation
- buyer lead status workflow
- evidence display
- feedback capture
- conservative deduplication behavior
- suggested next action

## Out of Scope
- Buyer Follow-Up Queue
- resizable split-pane layout
- autonomous buying
- automatic scam/fraud judgments
- replacing buyer judgment
- broad unrelated UI rewrites outside sourcing
